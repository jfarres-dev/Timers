# Capturas para el README

Coloca aquí los PNG con estos nombres exactos (son los que enlaza el `README.md` principal):

| Archivo | Qué capturar |
|---|---|
| `board.png` | El tablero con un par de columnas y varios timers (idealmente con etiquetas y algún 📌 tiempo fijo). |
| `card-config.png` | El modal de configuración de un timer con el check **📌 Tiempo fijo** marcado y el editor de horas visible. |
| `report.png` | El modal **📊 Informe** abierto, mostrando el desglose por columnas. |
| `daily-report.png` | El modal **📅 Días** abierto, mostrando el desglose por días. |

Consejos:
- Usa el navegador en una ventana ancha (escritorio) para que se vean varias columnas.
- Puedes hacerlas en modo claro u oscuro; recortar al área útil queda más limpio.
